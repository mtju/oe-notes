# mergeprop tool

Creates backups of _.properties_ files by default.

### Switches

 * -type
   * ubroker _(ubroker.properties)_
   * database _(conmgr.properties)_
   * tools _(JavaTools.properties)_
   * plugin _(AdminServerPlugins.properties)_
 * -action
   * update _(default)
   * create
   * delete
